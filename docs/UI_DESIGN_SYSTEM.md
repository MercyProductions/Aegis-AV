# UI Design System

The Aegis UI should feel premium, industrial, cinematic, alive, responsive, technical, and modern.

## Visual Rules

- Dark navy surfaces with silver borders.
- Electric blue accents for active state and trusted telemetry.
- Amber for review-needed security states.
- Glass panels with restraint.
- Smooth, short transitions.
- Dense information without clutter.
- Icon-first controls with tooltips where needed.

## Avoid

- Generic antivirus patterns.
- Bright clutter.
- Cheap gradients.
- Overcrowded screens.
- Decorative visuals that obscure security information.
