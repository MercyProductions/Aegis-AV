# Security Layers

Aegis uses layered protection instead of a single scanner verdict.

1. File Detection
2. Heuristic Analysis
3. Behavioral Monitoring
4. Process Monitoring
5. Script Monitoring
6. Ransomware Protection
7. Network Visibility
8. System Integrity Checks
9. User Awareness

The UI displays layer activity and health. Deductions in the security score should map back to one or more layers so the user can see why a score changed.
