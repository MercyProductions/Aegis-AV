# Distributed Architecture

Aegis is designed for local-first operation with optional scale-out.

## Modes

- Local-first operation.
- Optional cloud sync.
- Enterprise multi-device management.
- Edge processing.
- Modular deployments.
- Offline mode.

## Privacy Rule

Analyze locally first. Sync summaries, settings, policies, licenses, and metadata only when enabled. Do not upload personal files by default.
