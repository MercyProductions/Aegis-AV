# Live System Graph

The live system graph models the machine as a relationship map.

## Nodes

- Processes
- Services
- Drivers
- Connections
- Modules
- Startup entries
- Scheduled tasks
- Threats
- Automation events

## Features

- Zoom and filtering.
- Dependency visualization.
- Relationship tracing.
- Timeline replay.
- Risk overlays.

The graph is a visibility layer, not a hidden control channel.
