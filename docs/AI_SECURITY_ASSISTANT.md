# AI Security Assistant

The Aegis assistant explains security decisions. It does not independently block, quarantine, delete, or alter system settings.

## Capabilities

- Explain detections.
- Summarize incidents.
- Explain settings.
- Recommend next steps.
- Explain logs.
- Guide troubleshooting.

## Guardrails

- AI output is advisory.
- Blocking decisions stay in policy and detection engines.
- Personal files are not uploaded by default.
- The assistant should reduce panic by using calm, evidence-based language.
