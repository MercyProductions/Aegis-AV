# Brand Identity

Aegis should feel industrial, technical, premium, modern, confident, and clean.

## Voice

- Calm.
- Evidence-first.
- User-controlled.
- Clear about risk without panic.

## Visual Language

- Dark navy foundation.
- Silver separators.
- Electric blue status light.
- Glass command surfaces.
- Subtle motion.
- No clutter or generic antivirus noise.

## Product Language

- Security Center
- Protection Layers
- Incident Report
- Trust Controls
- Reliability Engineering
- Security Operations Center
